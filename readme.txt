== Description ==

Allows you to place new posts in a queue for later publication at a specified interval.

== Installation ==

1. Upload the folder `queue-posts` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Whenever you create a new post or page, you'll have the option to queue it for later
